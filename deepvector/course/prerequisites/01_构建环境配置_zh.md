# 构建环境配置

> 本教程所有章节共享的构建环境。首次配置后，后续章节直接引用本文档。

## 系统要求

| 组件 | 最低版本 | 推荐版本 | 说明 |
|------|----------|----------|------|
| OS | Ubuntu 20.04 / WSL2 | Ubuntu 22.04 | Linux 内核 5.4+ |
| g++ | g++-11 | g++-12 | 支持 C++17/C++20 |
| CMake | 3.16 | 3.25+ | 构建系统生成器 |
| Ninja | 1.10 | 1.12+ | 高速构建工具 |
| Python | 3.8 | 3.10+ | 绑定和脚本 |
| Git | 2.20 | 2.40+ | 版本控制 |

## 安装依赖

```bash
# Ubuntu / WSL2
sudo apt update
sudo apt install -y g++-12 cmake ninja-build python3 python3-pip git

# 验证版本
g++-12 --version
cmake --version
ninja --version
```

## CMake 基础

### 核心概念

| 概念 | 说明 |
|------|------|
| `cmake_minimum_required` | 声明最低 CMake 版本 |
| `project()` | 定义项目名称和语言 |
| `add_executable()` | 定义可执行目标 |
| `add_library()` | 定义库目标（静态/动态） |
| `target_link_libraries()` | 链接依赖库 |
| `target_include_directories()` | 添加头文件搜索路径 |
| `FetchContent` | 自动下载依赖（推荐方式） |

### 常用构建命令

```bash
# 配置（Debug 模式，带测试）
cmake -B build -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DENABLE_TESTS=ON \
  -DCMAKE_CXX_COMPILER=g++-12

# 配置（Release 模式，优化）
cmake -B build -G Ninja \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_CXX_COMPILER=g++-12

# 构建
cmake --build build -j$(nproc)

# 运行测试
ctest --test-dir build --output-on-failure
```

### 构建类型

| 类型 | 优化 | 调试信息 | 用途 |
|------|------|----------|------|
| `Debug` | 无 (-O0) | 完整 (-g) | 开发调试 |
| `Release` | 完整 (-O2/-O3) | 无 | 生产部署 |
| `RelWithDebInfo` | 部分 (-O2) | 部分 (-g) | 性能分析 |
| `MinSizeRel` | 体积优先 (-Os) | 无 | 嵌入式 |

### 常用 CMake 变量

```bash
-DCMAKE_BUILD_TYPE=Release      # 构建类型
-DCMAKE_CXX_COMPILER=g++-12     # 指定编译器
-DENABLE_TESTS=ON               # 启用测试
-DENABLE_PYTHON=ON              # 启用 Python 绑定
-DENABLE_BENCHMARKS=ON          # 启用基准测试
-DCMAKE_EXPORT_COMPILE_COMMANDS=ON  # 生成 compile_commands.json
```

## Ninja vs Make

| | Ninja | Make |
|---|---|---|
| 速度 | 更快（并行优化） | 较慢 |
| 增量构建 | 更精确 | 较粗 |
| 配合 CMake | `cmake -G Ninja` | `cmake -G "Unix Makefiles"` |
