# 向量距离度量 / Vector Distance Metrics

## L2 距离 (欧氏距离)

```cpp
float l2_squared(const float* a, const float* b, size_t dim) {
    float sum = 0;
    for (size_t i = 0; i < dim; i++) {
        float diff = a[i] - b[i];
        sum += diff * diff;
    }
    return sum;
}
```

## 内积 / Inner Product

```cpp
float inner_product(const float* a, const float* b, size_t dim) {
    float sum = 0;
    for (size_t i = 0; i < dim; i++) sum += a[i] * b[i];
    return -sum;  // 取负值使"越小越相似"
}
```

## 余弦相似度 / Cosine Similarity

```cpp
float cosine_distance(const float* a, const float* b, size_t dim) {
    float dot = 0, na = 0, nb = 0;
    for (size_t i = 0; i < dim; i++) {
        dot += a[i] * b[i];
        na += a[i] * a[i];
        nb += b[i] * b[i];
    }
    return 1 - dot / (std::sqrt(na) * std::sqrt(nb));
}
```
