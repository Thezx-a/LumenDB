# How to Use This Course

> Inspired by [Hello-Agents](https://github.com/datawhalechina/hello-agents): **small steps, each runnable.**  
> Goal: with basic C++/Python, you can build and run DeepVector end to end.

---

## 1. What you need

| Item | Minimum | Notes |
|------|---------|-------|
| OS | Linux / macOS / **Windows + WSL2** | C++ server uses POSIX sockets; native MSVC won't build it |
| Compiler | g++12+ or Apple Clang | C++17 libs, C++20 in parts of the server |
| Build | CMake ≥ 3.20, Ninja | See `prerequisites/01` |
| Python | 3.11+ | Required for Track B (Agent) |
| Ollama (optional) | local LLM | Skip for C++-only track |

**Free APIs (no paid LLM):** [FREE_RESOURCES_en.md](FREE_RESOURCES_en.md)

Runbook: [RUN.md](../../RUN.md) · Tech choices: [TECH.md](../../TECH.md)

## 2. Point → Line → Surface

Think LEGO:

| Level | You do | Example |
|-------|--------|---------|
| **Point** | One concept + tiny demo | `l2_squared` passes a unit test |
| **Line** | Wire two modules | `HNSW.search` + distance + `VectorStore.get` |
| **Surface** | Full Q&A path | `/ask` → embed → `/search` → LLM answer |

Don't skip chapters and paste code—you'll run it but won't explain it.

---

## 3. Pick a route

- **🟢 ~12h:** Track A ch01–03 + Track B overview + curl `/search`  
- **🟡 ~40h:** Full Track A + Track B through ch09 + Docker  
- **🔴 ~70h:** Both tracks + [INTERVIEW_BANK.md](INTERVIEW_BANK.md) + Capstone  

**Important:** Track A `ch04` (mmap) ≠ Track B `ch04` (LLM). Same number, different topics.

---

## 4. Common errors

| Symptom | Check |
|---------|-------|
| dim mismatch | server `--dim` = embedding dim (default **384**) |
| empty filter results | inserts need `meta.tags` |
| Windows build fails | WSL2 or Docker |
| empty Agent answers | run `scripts/demo_data.py` first |
