## 1. Point — Formulas & Syntax

| Metric | Formula idea | Code note |
|--------|--------------|-----------|
| L2² | sum of squared diffs | skip `sqrt` for ranking |
| IP-as-distance | negate dot product | fit “smaller is better” |
| Cosine distance | `1 - cos` | needs norms |

```cpp
inline float l2_squared_scalar(const float* a, const float* b, Dimension d) {
    float sum = 0.f;
    for (Dimension i = 0; i < d; ++i) {
        float diff = a[i] - b[i];
        sum += diff * diff;
    }
    return sum;
}
```

AVX2 sketch: `loadu` 8 floats → `sub` → `fmadd` accumulate. Enabled via `-mavx2 -mfma`.

**Source:** `include/dv/index/distance.h`
