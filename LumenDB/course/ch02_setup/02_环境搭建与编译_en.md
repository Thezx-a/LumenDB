## 2.1 Clone the Repository

```bash
git clone --recursive https://github.com/Thezx-a/LumenDB.git
cd LumenDB
```

If you forgot `--recursive`:

```bash
git submodule update --init --recursive
```

---

## 2.2 Compile C++ Server

```bash
cmake -B build -G Ninja \
    -DCMAKE_BUILD_TYPE=Release \
    -DCMAKE_CXX_COMPILER=g++-12 \
    -DUSE_AVX2=ON

cmake --build build -j$(nproc) --target lumendb_server
```

> **Line by line**:
> - `-B build`: Set build directory
> - `-G Ninja`: Use Ninja (faster than Make)
> - `-DUSE_AVX2=ON`: Enable AVX2 SIMD acceleration
> - `-j$(nproc)`: Parallel build using all CPU cores

Verify:

```bash
./build/server/lumendb_server --help
```

Expected output: shows `--port`, `--dim`, `--data-dir` options.

## 2.3 Install Python Dependencies

```bash
# Minimal install
pip install httpx pydantic sentence-transformers

# Full install (recommended)
pip install fastapi uvicorn mcp pytest pytest-asyncio
```

Verify:

```bash
python -c "from agent.config import load_config; print(load_config())"
```

---

## 2.4 Install Ollama (Optional)

```bash
curl -fsSL https://ollama.com/install.sh | sh
ollama pull nomic-embed-text   # embedding (~274MB)
ollama pull qwen2.5:3b         # light LLM (~1.7GB)
ollama serve
```

---

## 2.5 Verify the System

```bash
# Terminal 1: Start LumenDB
./build/server/lumendb_server --port 8080 --dim 384

# Terminal 2: Health check
curl http://localhost:8080/health

# Terminal 3: Start Agent
python -m agent.server.app
```
