# Build Environment Configuration

> The build environment shared across all chapters of this tutorial. After the initial configuration, subsequent chapters reference this document directly.

## System Requirements

| Component | Minimum Version | Recommended Version | Notes |
|-----------|----------------|---------------------|-------|
| OS | Ubuntu 20.04 / WSL2 | Ubuntu 22.04 | Linux kernel 5.4+ |
| g++ | g++-11 | g++-12 | Supports C++17/C++20 |
| CMake | 3.16 | 3.25+ | Build system generator |
| Ninja | 1.10 | 1.12+ | High-speed build tool |
| Python | 3.8 | 3.10+ | Bindings and scripts |
| Git | 2.20 | 2.40+ | Version control |

## Installing Dependencies

```bash
# Ubuntu / WSL2
sudo apt update
sudo apt install -y g++-12 cmake ninja-build python3 python3-pip git

# Verify versions
g++-12 --version
cmake --version
ninja --version
```

## CMake Fundamentals

### Core Concepts

| Concept | Description |
|---------|-------------|
| `cmake_minimum_required` | Declares the minimum CMake version |
| `project()` | Defines the project name and languages |
| `add_executable()` | Defines an executable target |
| `add_library()` | Defines a library target (static/dynamic) |
| `target_link_libraries()` | Links dependency libraries |
| `target_include_directories()` | Adds header file search paths |
| `FetchContent` | Automatically downloads dependencies (recommended approach) |

### Common Build Commands

```bash
# Configure (Debug mode, with tests)
cmake -B build -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DENABLE_TESTS=ON \
  -DCMAKE_CXX_COMPILER=g++-12

# Configure (Release mode, optimized)
cmake -B build -G Ninja \
  -DCMAKE_BUILD_TYPE=Release \
  -DCMAKE_CXX_COMPILER=g++-12

# Build
cmake --build build -j$(nproc)

# Run tests
ctest --test-dir build --output-on-failure
```

### Build Types

| Type | Optimization | Debug Info | Use Case |
|------|-------------|------------|----------|
| `Debug` | None (-O0) | Full (-g) | Development and debugging |
| `Release` | Full (-O2/-O3) | None | Production deployment |
| `RelWithDebInfo` | Partial (-O2) | Partial (-g) | Profiling |
| `MinSizeRel` | Size-optimized (-Os) | None | Embedded |

### Common CMake Variables

```bash
-DCMAKE_BUILD_TYPE=Release      # Build type
-DCMAKE_CXX_COMPILER=g++-12     # Specify compiler
-DENABLE_TESTS=ON               # Enable tests
-DENABLE_PYTHON=ON              # Enable Python bindings
-DENABLE_BENCHMARKS=ON          # Enable benchmarks
-DCMAKE_EXPORT_COMPILE_COMMANDS=ON  # Generate compile_commands.json
```

## Ninja vs Make

| | Ninja | Make |
|---|---|---|
| Speed | Faster (parallel optimization) | Slower |
| Incremental builds | More precise | Coarser |
| CMake integration | `cmake -G Ninja` | `cmake -G "Unix Makefiles"` |
