# 测试框架

> 本教程使用 Google Test 进行单元测试。本文档覆盖测试基础。

## Google Test 基础

### 测试结构

```cpp
#include <gtest/gtest.h>

// 测试夹具（共享 setup/teardown）
class MyTest : public ::testing::Test {
protected:
    void SetUp() override { /* 每个测试前执行 */ }
    void TearDown() override { /* 每个测试后执行 */ }
    int value = 42;
};

// 断言宏
TEST_F(MyTest, BasicAssertion) {
    EXPECT_EQ(value, 42);       // 非致命断言
    ASSERT_TRUE(value > 0);     // 致命断言（失败则终止当前测试）
    EXPECT_NEAR(3.14, 3.14, 0.001);  // 浮点近似比较
}

// 值参数化测试
class ParameterizedTest : public ::testing::TestWithParam<int> {};

TEST_P(ParameterizedTest, Works) {
    EXPECT_GT(GetParam(), 0);
}

INSTANTIATE_TEST_SUITE_P(
    PositiveNumbers,
    ParameterizedTest,
    ::testing::Values(1, 5, 100)
);
```

### 常用断言

| 断言 | 说明 | 致命性 |
|------|------|--------|
| `EXPECT_EQ(a, b)` | a == b | 非致命 |
| `ASSERT_EQ(a, b)` | a == b | 致命 |
| `EXPECT_TRUE(expr)` | expr 为真 | 非致命 |
| `ASSERT_TRUE(expr)` | expr 为真 | 致命 |
| `EXPECT_NEAR(a, b, eps)` | \|a-b\| < eps | 非致命 |
| `EXPECT_THROW(stmt, Ex)` | 抛出异常 | 非致命 |

### 运行测试

```bash
# 运行所有测试
ctest --test-dir build --output-on-failure

# 运行特定测试
./build/tests/test_name

# 过滤测试
./build/tests/test_name --gtest_filter="TestSuite.TestCase"

# 查看所有测试
./build/tests/test_name --gtest_list_tests
```

### CMake 集成

```cmake
# 启用测试
enable_testing()
find_package(GTest REQUIRED)

# 添加测试
add_executable(test_name test_file.cpp)
target_link_libraries(test_name GTest::gtest_main)

# 注册到 CTest
add_test(NAME test_name COMMAND test_name)
```
