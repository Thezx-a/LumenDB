# 向量距离度量

> 向量数据库的核心操作。本文档覆盖常用距离度量的定义和选择。

## 三种主要度量

### L2 距离（欧几里得距离）

$$d(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}$$

- **几何含义**：两点之间的直线距离
- **适用场景**：图像特征、物理坐标
- **特点**：对向量长度敏感

### 余弦相似度

$$\text{cosine\_sim}(\mathbf{x}, \mathbf{y}) = \frac{\mathbf{x} \cdot \mathbf{y}}{\|\mathbf{x}\| \|\mathbf{y}\|}$$

- **几何含义**：两个向量夹角的余弦值
- **适用场景**：文本搜索、语义匹配
- **特点**：只关心方向，不关心长度

### 内积（点积）

$$\text{inner\_product}(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^{n} x_i \cdot y_i$$

- **几何含义**：一个向量在另一个方向上的投影
- **适用场景**：推荐系统、注意力机制
- **特点**：值越大越相似

## 选择指南

| 应用场景 | 推荐度量 | 原因 |
|----------|----------|------|
| 文本搜索 | 余弦 | 归一化后只关心语义方向 |
| 图像检索 | L2 | 像素级差异重要 |
| 推荐系统 | 内积 | 用户/物品向量长度有含义 |
| 通用语义 | 余弦 | 最常用，鲁棒性好 |

## LumenDB 中的实现

```cpp
// L2 距离
float l2_distance(const float* a, const float* b, int dim) {
    float sum = 0;
    for (int i = 0; i < dim; ++i) {
        float diff = a[i] - b[i];
        sum += diff * diff;
    }
    return std::sqrt(sum);
}

// 余弦距离 = 1 - cosine_similarity
float cosine_distance(const float* a, const float* b, int dim) {
    float dot = 0, norm_a = 0, norm_b = 0;
    for (int i = 0; i < dim; ++i) {
        dot += a[i] * b[i];
        norm_a += a[i] * a[i];
        norm_b += b[i] * b[i];
    }
    return 1.0f - dot / (std::sqrt(norm_a) * std::sqrt(norm_b));
}
```
