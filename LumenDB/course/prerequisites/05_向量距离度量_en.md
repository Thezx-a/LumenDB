# Vector Distance Metrics

> The core operation of vector databases. This document covers common distance metric definitions and selection guidance.

## Three Main Metrics

### L2 Distance (Euclidean Distance)

$$d(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}$$

- **Geometric meaning**: The straight-line distance between two points
- **Use cases**: Image features, physical coordinates
- **Characteristics**: Sensitive to vector magnitude

### Cosine Similarity

$$\text{cosine\_sim}(\mathbf{x}, \mathbf{y}) = \frac{\mathbf{x} \cdot \mathbf{y}}{\|\mathbf{x}\| \|\mathbf{y}\|}$$

- **Geometric meaning**: The cosine of the angle between two vectors
- **Use cases**: Text search, semantic matching
- **Characteristics**: Only cares about direction, not magnitude

### Inner Product (Dot Product)

$$\text{inner\_product}(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^{n} x_i \cdot y_i$$

- **Geometric meaning**: The projection of one vector onto the direction of another
- **Use cases**: Recommendation systems, attention mechanisms
- **Characteristics**: Higher values indicate greater similarity

## Selection Guide

| Application Scenario | Recommended Metric | Reason |
|---------------------|-------------------|--------|
| Text search | Cosine | After normalization, only semantic direction matters |
| Image retrieval | L2 | Pixel-level differences are important |
| Recommendation systems | Inner product | User/item vector magnitudes are meaningful |
| General semantics | Cosine | Most commonly used, good robustness |

## Implementation in LumenDB

```cpp
// L2 distance
float l2_distance(const float* a, const float* b, int dim) {
    float sum = 0;
    for (int i = 0; i < dim; ++i) {
        float diff = a[i] - b[i];
        sum += diff * diff;
    }
    return std::sqrt(sum);
}

// Cosine distance = 1 - cosine_similarity
float cosine_distance(const float* a, const float* b, int dim) {
    float dot = 0, norm_a = 0, norm_b = 0;
    for (int i = 0; i < dim; ++i) {
        dot += a[i] * b[i];
        norm_a += a[i] * a[i];
        norm_b += b[i] * b[i];
    }
    return 1.0f - dot / (std::sqrt(norm_a) * std::sqrt(norm_b));
}
```
