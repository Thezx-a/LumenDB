# Testing Framework

> This tutorial uses Google Test for unit testing. This document covers testing fundamentals.

## Google Test Basics

### Test Structure

```cpp
#include <gtest/gtest.h>

// Test fixture (shared setup/teardown)
class MyTest : public ::testing::Test {
protected:
    void SetUp() override { /* Executed before each test */ }
    void TearDown() override { /* Executed after each test */ }
    int value = 42;
};

// Assertion macros
TEST_F(MyTest, BasicAssertion) {
    EXPECT_EQ(value, 42);       // Non-fatal assertion
    ASSERT_TRUE(value > 0);     // Fatal assertion (terminates current test on failure)
    EXPECT_NEAR(3.14, 3.14, 0.001);  // Floating-point approximate comparison
}

// Value-parameterized test
class ParameterizedTest : public ::testing::TestWithParam<int> {};

TEST_P(ParameterizedTest, Works) {
    EXPECT_GT(GetParam(), 0);
}

INSTANTIATE_TEST_SUITE_P(
    PositiveNumbers,
    ParameterizedTest,
    ::testing::Values(1, 5, 100)
);
```

### Common Assertions

| Assertion | Description | Fatality |
|-----------|-------------|----------|
| `EXPECT_EQ(a, b)` | a == b | Non-fatal |
| `ASSERT_EQ(a, b)` | a == b | Fatal |
| `EXPECT_TRUE(expr)` | expr is true | Non-fatal |
| `ASSERT_TRUE(expr)` | expr is true | Fatal |
| `EXPECT_NEAR(a, b, eps)` | \|a-b\| < eps | Non-fatal |
| `EXPECT_THROW(stmt, Ex)` | Throws an exception | Non-fatal |

### Running Tests

```bash
# Run all tests
ctest --test-dir build --output-on-failure

# Run a specific test
./build/tests/test_name

# Filter tests
./build/tests/test_name --gtest_filter="TestSuite.TestCase"

# List all tests
./build/tests/test_name --gtest_list_tests
```

### CMake Integration

```cmake
# Enable testing
enable_testing()
find_package(GTest REQUIRED)

# Add a test
add_executable(test_name test_file.cpp)
target_link_libraries(test_name GTest::gtest_main)

# Register with CTest
add_test(NAME test_name COMMAND test_name)
```
